Instrucciones

  • Usa las flechas izquierda y derecha para mover la plataforma y posicionarla estratégicamente.
  • Presiona la flecha arriba para lanzar la bola y comenzar el juego.
  • Tienes 3 vidas para completar todas las fases, así que úsalas sabiamente.
  • Destruye todos los bloques para avanzar a la siguiente fase y enfrentar un nuevo reto.
